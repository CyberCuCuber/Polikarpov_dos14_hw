ping - команда для проверки качества соединения с каким-либо узлом. Использует ICMP echo request. С помощью этой команды можно процерить доступность и целостность отправляемых пакетов.
traceroute - команда для определения маршрута следования пакетов до конечного узла. Может быть использована для тестирования сети на доступность и поиска неработающих узлов.
nslookup - команда для обращения к dns-узлу. Можно использовать для проверки на наличие необходимых dns-записей на сервере.
dig - команда, аналогичная nslookup, которая имеет более развёрнутую информацию о запросе и ответе. Может использоваться для поиска информации о домене, такой как dns-записи, ip, а так же наличие конкретной записи на конкретном dns-сервере
ss - команда для вывода статистики сетевых соединений. Может использоваться для проверки интернет соединений компьютера/сервера
nmap - команда для сканирования открытых/закрытых портов на конкретном узле. Так же можно получить информацию об интерфейсах на портах. Команда гибко настраивается для конкретных заданий.
